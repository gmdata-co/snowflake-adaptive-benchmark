# Adapted TPC-H Queries for Snowflake

These are TPC-H benchmark queries (q01.sql - q22.sql) that have been adapted for Snowflake.

## Source

These queries are generated by the `adapt_queries.py` script from the original TPC-H queries in the `original_queries` folder.

## Adaptations Applied

The following transformations are applied to make the queries compatible with Snowflake:

1. **Metadata removal**: Lines like `:x`, `:o`, `:n` are removed
2. **Parameter substitution**: Placeholders (`:1`, `:2`, etc.) are replaced with actual values
3. **Table qualification**: Table names are prefixed with `SNOWFLAKE_SAMPLE_DATA.TPCH_SF100`
4. **Syntax fixes**:
   - Interval syntax converted to Snowflake format
   - Substring syntax converted to SUBSTR function
   - Query 15's CREATE VIEW pattern converted to CTE (WITH clause)

## Usage

These queries are used by the benchmark scripts to measure Snowflake query performance.
